# Product Vision

Schengen90 helps travellers understand and plan their legal stay within the Schengen Area.

The application focuses on one goal:

Calculate remaining stay accurately using the official rolling 90/180-day rule while making the result easy to understand.

The application should always prioritise:

- Accuracy
- Simplicity
- Transparency
- Privacy

The application should never require an internet connection for calculations.

All travel history should remain on the user's device unless they explicitly choose to export it.
